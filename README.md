# Node.js + React 的聊天系统
## 简介
这是一个使用 Express.js 和 React.js 构建的实时聊天系统。该项目展示了如何结合前端和后端技术来创建一个简单的聊天应用，包括用户认证、消息实时更新等功能。本项目适合初学者作为学习和实践的示范项目。

## 功能

- 登录和注册
- 聊天室列表：查看、新建、编辑聊天室
- 聊天室：可以实时收发消息
- 用户信息管理

## 技术栈
- **前端**: React, Typescript, Material UI, Redux, Axios
- **后端**: Express, jsonwebtoken, Mariadb, Sequelize

## 安装与运行

### 前置条件
- Node.js
- MySql 或 Mariadb 数据库

### 运行步骤
- 安装依赖
```
yarn install
```

- 运行
```
yarn start
```
